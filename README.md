* [BEM Methodology](https://en.bem.info/methodology/)
* [JavaScript](https://en.wikipedia.org/wiki/JavaScript)
* [React](https://en.wikipedia.org/wiki/React_(JavaScript_library))
* [Webpack](https://webpack.js.org/)
